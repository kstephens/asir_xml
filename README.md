XML Coder for ASIR

## Installation

Add this line to your application's Gemfile:

    gem 'asir_xml'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install asir_xml

## Usage

See example/.

## Dependencies

Requires install of libxml, libxslt for XML support: rake prereq.
